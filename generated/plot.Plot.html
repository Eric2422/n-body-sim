plot.Plot¶

class plot.Plot(data_frame: DataFrame, time_step_size: float = 1.0, margin: float = 1.25, min: float = 1.0)¶

Bases: object

A Matplotlib plot of the movement of particles over time. The plot will be run in real-time (i.e., each second of the simulation will be displayed as a second of animation).

Parameters:

  • data_frame (pandas.DataFrame) -- A data frame with four columns: t, x, y, and z. Contains the time and position of particles.

  • time_step_size (float, default=1.0) -- The amount of time between each frame.

  • margin (float, default=1.25) --

    The amount of extra space in each dimension as a factor of its size.

    For example, if the x dimension has a range of [-1.0, 1.0], a margin of 1.25 means that the plot will have an x range of [-1.25, 1.25].

  • min (float, default=1.0) -- The minimum size of each dimension.

Save the plot as a GIF file with the given name. The filename will be resolved using files.FileHandler.OUTPUT_DIR, i.e., the GIF will be saved under the same directory as the text files.

Parameters:

filename (str) -- The filename that the GIF will be saved to. It does not matter if it includes a file extension or what file extension is used. Any file extension will be removed and replaced with .gif.
