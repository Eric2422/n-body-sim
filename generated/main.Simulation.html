main.Simulation¶

class main.Simulation(theta: float = 0.5, time_step_size: float = 1.0, gravitational_field: FieldVector = array([0., 0., 0.]), electric_field: FieldVector = array([0., 0., 0.]), magnetic_field: FieldVector = array([0., 0., 0.]), particles_list: list[PointParticle] = [])¶

Bases: object

One simulation of a given initial conditions of particles and fields.

Contains the Barnes-Hut approximation parameter; time step size; constant, uniform gravitational field; constant, uniform electrical field; constant, uniform magnetic field; and particles used in the simulation.

Keeps track of the previous positions of all the partices.

Parameters:

  • theta (float, default=0.5) -- The Barnes-Hut approximation parameter.

  • time_step_size (float, default=1.0) -- The time increment of the simulation in seconds (s).

  • gravitational_field (vectors.FieldVector, default = numpy.zeros() (3, dtype=float)) -- A constant, uniform gravitational field.

  • electric_field (vectors.FieldVector, default = numpy.zeros() (3, dtype=float)) -- A constant, uniform electric field.

  • magnetic_field (vectors.FieldVector, default = numpy.zeros() (3, dtype=float)) -- A constant, uniform magnetic field.

  • particles_list (list [particles.PointParticle], default = []) -- A list of particles that are interacting with each other in the simulation.
