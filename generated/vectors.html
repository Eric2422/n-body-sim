vectors¶

Module containing type aliases for 1 × 3 NumPy arrays. Strictly speaking, they serve no actual purpose. They exist mainly to better specify what an array is supposed to be used for, essentially dimensional analysis.

Module Attributes

UnitVector

A 3D unit vector that represents a direction.

DisplacementVector

A 3D vector that represents the displacement between two points.

PositionVector

A 3D vector that represents a point.

VelocityVector

A 3D vector that represents a velocity.

AccelerationVector

A 3D vector that represents an acceleration.

FieldVector

A 3D vector that represents a field at a point.

ForceVector

A 3D vector that represents a force.
