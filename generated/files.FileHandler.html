files.FileHandler¶

class files.FileHandler(schema_file: str = 'main.json', input_filepath: str = 'sample.json')¶

Bases: object

Class of attributes and methods to create, read, and write to files.

Parameters:

  • schema_file (str, default="schema.json") --

    The name of the JSON schema file used for the input files.

    The file name will automatically be resolved with respect to FileHandler.INPUT_DIR, so it should not include the directory at the beginning. Best to keep it to the default unless you want to write an entire custom JSON schema.

  • input_file (str, default="sample.csv") --

    The filepath of the input file, including file extension.

    Accepts both with and without the directory. The output file will have the same name but with the ".txt" file extension instead.

append_to_output_file(output_string: str = '\n') → None¶

Append the given string into the output file. If OUTPUT_FILE_PATH has already been opened, then the string will be appended to it without closing.

Elsewise, it will open OUTPUT_FILE_PATH, append to it, and close it.

write_input_file(input_dict: dict) → None¶

Write a schema-valid Python dictionary into a input JSON file.

The file must be in INPUT_DIR. The input_dict must conform to the JSON schemas in SCHEMA_DIR.

If the file does not exist, a new file will be created. If the file does exist, any pre-existing content will be overwritten. The file will have the same name as INPUT_FILE_PATH.

Parameters:

input_dict (dict) -- An object to write into the file as a JSON.
