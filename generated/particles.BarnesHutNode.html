particles.BarnesHutNode¶

class particles.BarnesHutNode(particles: list[PointParticle] = [], x_bounds: ndarray[tuple[Any, ...], dtype[float64]] | None = None, y_bounds: ndarray[tuple[Any, ...], dtype[float64]] | None = None, z_bounds: ndarray[tuple[Any, ...], dtype[float64]] | None = None)¶

Bases: object

A single node of a Barnes-Hut octree, which contains eight child nodes. Assumes a center of charge rather than using a multipole expansion.

Notes

The implementation of the Barnes-Hut algorithm is taken from [1].

Parameters:

  • particles (list [particles.PointParticle], default=[]) -- List of particles that are contained within this Barnes-Hut node.

  • x_bounds (numpy.typing.NDArray [numpy.float64], optional) -- A two-element NumPy array that contains the lower and upper x bounds in that order. If None, the bounds will be automatically calculated to be the smallest possible that would contain all the particles.

  • y_bounds (numpy.typing.NDArray [numpy.float64], optional) -- A two-element NumPy array that contains the lower and upper y bounds in that order. If None, the bounds will be automatically calculated to be the smallest possible that would contain all the particles.

  • z_bounds (numpy.typing.NDArray [numpy.float64], optional) -- A two-element NumPy array that contains the lower and upper z bounds in that order. If None, the bounds will be automatically calculated to be the smallest possible that would contain all the particles.

get_electric_field_exerted(point: PositionVector, theta: float = 0.0, particle_id: int = -1) → FieldVector¶

Calculate the approximate electric field exerted by this node at a given point.

Parameters:

  • point (vectors.PositionVector) -- The position to calculate the electric field at. Measured in meters (m).

  • theta (float, default=0.0) --

    The value of theta, the Barnes-Hut approximation parameter being used. Given the distance between the point and the center of charge, it used to determine whether to return an approximate or exact value for the gravitational field.

    When 0.0, no approximation will occur.

  • particle_id (int, default=-1) --

    The ID of the particle to exclude from the force calculation.

    When -1, no particles will be excluded from the force calculation.

Returns:

The electric field vector produced by this node measured in newtons per coulomb (N/C).

Return type:

vectors.FieldVector
